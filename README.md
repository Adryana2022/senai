# senai
Atividade unidade codificação back end / Senai
sistema de controle de clientes, em que é possível aplicar os seguintes elementos da linguagem C#:
Entrada e saída de dados;
Declaração de variáveis.
Operadores aritméticos;
Estruturas condicionais e de repetição se necessário;
Operadores relacionais e lógicos.
Também aplica conceitos de Programação Orientada a Objetos – POO:
Abstração;
Classes, propriedades e métodos;
Herança;
Polimorfismo;
Encapsulamento.
